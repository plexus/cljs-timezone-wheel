# timezone-wheel

ClojureScript/Reagent port of the very cool
[Timezone Wheel](https://codepen.io/daiyi/pen/YZrezB) by
[@dayitastic](https://twitter.com/daiyitastic).

## Try it!

Try it out with `lein figwheel`.
